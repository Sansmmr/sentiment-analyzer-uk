# Система аналізу настроїв текстів українською мовою
## Ukrainian Text Sentiment Analysis System

**Версія:** MVP 1.0  
**Автор:** Твердохлібов Данило Ігорович 
**Дата:** Травень 2026

---

## 📋 Короткий опис проєкту

Цей проєкт реалізує MVP системи аналізу настроїв (sentiment analysis) для текстів українською мовою. Система дозволяє автоматично визначати емоційне забарвлення тексту — позитивне, негативне або нейтральне.

### Ключова цінність:
- Швидкий аналіз текстів без використання зовнішніх API
- Повна автономність роботи
- Підтримка української мови
- Простота використання

---

## ⚙️ Основний функціонал

### 1. Аналіз настрою тексту
- Визначення позитивного, негативного та нейтрального настрою
- Розрахунок sentiment score (-1.0 до +1.0)
- Визначення впевненості аналізу (confidence)

### 2. Виявлення ключових слів
- Пошук позитивних слів у тексті
- Пошук негативних слів у тексті
- Врахування заперечень ("не добрий" → негатив)

### 3. Пакетна обробка
- Одночасний аналіз декількох текстів
- Експорт результатів

### 4. Підтримка української мови
- 300+ позитивних слів
- 300+ негативних слів
- Врахування морфології української мови

## 🚀 Інструкція користувача

### Встановлення

1. **Клонування репозиторію:**
```bash
git clone https://github.com/username/sentiment-analyzer-uk.git
cd sentiment-analyzer-uk
```

2. **Перевірка наявності Python:**
```bash
python --version  # Python 3.8 або вище
```

3. **Запуск демонстрації:**
```bash
python sentiment_analyzer.py
```

### Використання

**Базовий приклад:**
```python
from sentiment_analyzer import UkrainianSentimentAnalyzer

# Створення аналізатора
analyzer = UkrainianSentimentAnalyzer()

# Аналіз тексту
text = "Цей продукт чудовий! Я дуже задоволений."
result = analyzer.analyze(text)

print(f"Настрій: {result['sentiment']}")
print(f"Бали: {result['score']}")
print(f"Впевненість: {result['confidence']}")
```

**Пакетний аналіз:**
```python
texts = [
    "Чудовий сервіс!",
    "Жахливе обслуговування",
    "Нормальний продукт"
]
results = analyzer.analyze_batch(texts)
```

### Алгоритм роботи:
1. **Лексичний аналіз** — словниковий метод на основі списків позитивних/негативних слів
2. **Обробка заперечень** — інвертування настрою при наявності "не", "без"
3. **Посилювачі** — множителі для слів типу "дуже", "надзвичайно"
4. **Нормалізація** — приведення до нижнього регістру, очищення від спецсимволів

## ⚠️ Відомі обмеження системи

### Технічні обмеження:
1. **Словниковий метод** — не враховує контекст та іронію
2. **Обмежений словник** — базова версія має ~600 слів
3. **Без машинного навчання** — статичні правила аналізу

### Функціональні обмеження:
1. **Точність** — ~70-75% на коротких текстах
2. **Не розпізнає сарказм**
3. **Не враховує emoji** (можна додати)
4. **Без збереження історії аналізів**
5. **Тільки українська мова**

### Можливі покращення:
- Інтеграція з ML-моделями (BERT, FastText)
- Розширення словника до 10,000+ слів
- Веб-інтерфейс
- API для інтеграції
- Багатомовність
